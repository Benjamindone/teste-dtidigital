# teste-dtidigital
Repositório para teste da dti-digital

Este projeto foi desenvolvido como parte de um processo seletivo para estágio em desenvolvimento.

O sistema permite que um professor registre as notas e a frequência de seus alunos em cinco disciplinas. A aplicação calcula automaticamente a média das notas de cada aluno, a média da turma em cada disciplina e identifica alunos que necessitam de atenção especial.
